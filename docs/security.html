Security disclosure and reporting — AutoPromote

If you believe you have found a security vulnerability in AutoPromote, please report it to us so we can investigate and remediate it promptly.

How to report

Send an email to tmtshwelo21@gmail.com with the following information:

  • Short description of the issue.
  • Step-by-step reproduction steps, with URLs and sample requests if applicable.
  • Proof-of-concept or screenshots (redact sensitive user data).
  • Your contact email and a preferred time window for follow-up.

What we will do

  • Acknowledge receipt within 3 business days.
  • Confirm whether the issue is a valid security vulnerability.
  • Work to provide a fix and notify the reporter when the issue is resolved.

Public disclosure

Please do not publicly disclose any vulnerability until we have had a chance to investigate and remediate. We will work with you on coordinated disclosure if requested.
